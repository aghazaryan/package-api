This project is a backend server for managing packages, implemented using Node.js, TypeScript, NestJS, and PostgreSQL. 
It includes endpoints for CRUD operations on single and multiple packages, unit and integration tests, and a Swagger documentation page.

## Technologies Used
* Node.js: JavaScript runtime built on Chrome's V8 JavaScript engine.
* TypeScript: Typed superset of JavaScript that compiles to plain JavaScript.
* NestJS: A progressive Node.js framework for building efficient and scalable server-side applications.
* PostgreSQL: A powerful, open-source object-relational database system.

## Features
* CRUD operations for single and multiple packages.
* Pagination support for listing packages.
* Unit and integration tests for ensuring code quality.
* Swagger documentation for API endpoints.

## Getting Started
### Prerequisites
Make sure you have Docker and Docker Compose installed on your machine.

### Installation

1. Clone the repository:
```shell
git clone https://github.com/aghazaryan/package-api.git
cd package-api
```
2. Build and run the application using Docker Compose:
```shell
docker-compose up --build
```

## API Documentation
The Swagger documentation for the API is accessible at:
```shell
http://localhost:3000/swagger
```

This page provides details on all the available endpoints and how to use them.
